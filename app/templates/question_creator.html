Question Creator

In order to create a new instance of a question type, you need to do the Following steps

1 - Name the question type, The advice is to use the name + Question(e.g "FoodQuestion")

2 - Find the wikidata entity identifier, (e.g Q38723 - higher education institution)

3 - This initial implementation goal is to list all instances and sub-classes of the selected entity, but it can't predict weather the results are good or not. you can also add filters in the field

4 - Some SPARQL knowledge is advisable, after inserting the name and the entity identifier, you can query the RLWF triple-store to see if it was correctly created. Otherwise contact the system administraction for the deletion of the wrong value
