Question Creator

In order to create a new instance of a question type in the local triple store:

1 - Name the question similarly to the existing pattern name + Question(e.g "FoodQuestion").

2 - Find the wikidata entity identifier, (e.g Q38723 - higher education institution).

3 - This initial implementation goal is to list all instances of the selected entity identifier, but it can't predict weather the results are good or not. Filters can help.

4 - Some SPARQL knowledge is required, after inserting the name and the entity identifier, you can query the RLWF triple-store to see if it was correctly created.

5 - Some HTML knowledge is also required, the question explorer page does not support the dynamic listing of new questions. the elif function from line 131 to 142 in question_explorer.html needs to be replicated with the correct question values.
